# 🎮 Pokedevs List

> Uma pokédex temática de desenvolvedores, onde cada pokedev tem seu próprio tipo, número e descrição inspirada no universo da programação.

## 🔗 Demo

Acesse o projeto ao vivo: **[https://matheus-larre.github.io/projeto-pokedevs/](https://matheus-larre.github.io/projeto-pokedevs/)**

---

## 📖 Sobre o Projeto

O **Pokedevs List** é meu primeiro projeto web, desenvolvido para colocar em prática os conhecimentos iniciais de **HTML5**, **CSS3** e **JavaScript**. A ideia foi criar uma pokédex temática onde os "pokémons" são, na verdade, perfis de desenvolvedores com personalidades e habilidades únicas inspiradas no mundo do desenvolvimento de software.

---

## ✨ Funcionalidades

- Listagem de 6 pokedevs únicos, cada um com nome, número, tipo e descrição
- Navegação interativa entre os cards via barra lateral
- Layout responsivo adaptado para diferentes tamanhos de tela
- Estilização temática com cores por tipo (elétrico, fogo, água, fada)

---

| Nome | Nº | Tipo | Habilidade |

| Psycoder | #022 | 💧 Água | Sênior com habilidades extraordinárias sob pressão |
| Charmandev | #015 | 🔥 Fogo | Nunca deixa a chama do aprendizado se apagar |

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização, responsividade e reset de estilos
- **JavaScript** — interatividade na navegação entre cards
- **Google Fonts** — fonte Poppins

## 🚀 Como Executar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/matheus-larre/projeto-pokedevs.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd projeto-pokedevs
   ```

3. Abra o arquivo `index.html` no seu navegador.

> Não há dependências ou instalações necessárias — é um projeto front-end puro!
